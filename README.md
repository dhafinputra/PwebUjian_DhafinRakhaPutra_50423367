Aplikasi web sederhana berbasis CRUD (Create, Read, Update, Delete) yang digunakan untuk mengelola data quotes menggunakan PHP dan JavaScript.

# Teknologi yang Digunakan
HTML
CSS
JavaScript (Fetch API)
PHP (Native)
MySQL
# Struktur Folder
ujian/ ├── api/ │ ├── create.php │ ├── read.php │ ├── update.php │ ├── delete.php │ └── db.php ├── public/ │ ├── index.html │ ├── app.js │ └── style.css

# Cara Menjalankan Aplikasi
Pindahkan folder ujian ke dalam folder htdocs (XAMPP).
Jalankan Apache dan MySQL.
Import database ke phpMyAdmin.
Buka browser dan akses: http://localhost/ujian/public/index.html
